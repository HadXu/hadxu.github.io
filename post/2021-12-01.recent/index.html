12月近况

cross-compile

喜欢上了折腾NAS,好想赶紧回家搞NAS，总价2999，还是很香的。突然发现威联通好像不是很香啊，哈哈哈。

在折腾rust cross-compile的时候，突然想将mac上的编译放到linux上运行，好在手里有mac以及linux云服务器，因此就尝试了一下，亲测可行。

官方中说rust支持各种平台，可以使用rustup target list来看，如果想在哪个平台上运行就用rustup target xxx来添加，接下来使用cargo build --target=arm-linux-androideabi就可以编译啦，实际中mac端需要进行如下的操作

1
2
3
4

$ rustup target add x86_64-unknown-linux-gnu
$ brew tap SergioBenitez/osxct
$ brew install x86_64-unknown-linux-gnu
$ CARGO_TARGET_X86_64_UNKNOWN_LINUX_GNU_LINKER=x86_64-unknown-linux-gnu-gcc cargo build --release --target=x86_64-unknown-linux-gnu

亲测在linux端可以运行。

arm_neon折腾记

手里有个mac，折腾一下arm端代码的编写，这也属于异构计算？开始用的vscode进行c++的编写，后来发现还是CLion比较香。 ARM Neon Intrinsics 学习指北：从入门、进阶到学个通透看起来学习成本挺高的，不过值得研究。

突然看到消息，不确定是不是tx的员工跳楼了，哎，互联网的压力导致每个人都那么压抑，希望能够好好生活，健康生活就可以了。

没有想到过了一天，那个员工去世的消息大家也不再讨论，好像失去了热度，一切都回归正常，对于一个社会而言，一条生命又算得了什么呢？而对于一个家庭而言，这是多大的伤害啊。无能为力，想想自己的家庭，想想自己的亲戚，还是挺难过的，这就是我们想要的样子吗？我觉得不是，还是好好打工吧。

最近学习scoursera上面的cs课程，从逻辑电路到俄罗斯方块，目前看还是挺有意思的，希望能够坚持下去吧。还有看《os three piece》，看起来不错。

把操作系统的《os three piece》看完了，总结一下就是操作系统就是虚拟化、并发以及持久化的一个机器。对于虚拟化而言，CPU要虚拟化、内存条要虚拟化。对于并发而言，CPU能够并发，不同程序之间拥有各自的内存空间，然后在程序之内有不同的线程来共享一个内存空间，但是由于不需要保存数据以及地址，因此更加轻量化。对于持久化而言，有各种存储，终于把SSD原理搞明白了，佩服日本的半导体产业，还有韩国的三星公司制造的SSD，目前SSD的速度已经达到了2GB/s，还是很快的。

昨天公司年会，只有阳光普照66，😭。

最近把俄罗斯的cs课程学完，加油。

突然想到之前买的亚马逊的2年书籍，到现在就看了2本，不理智的消费呀，浪费了100多元，可惜了。

俄罗斯的cs课程真的非常amazing啊，目前12月28日学到了RAM的构造，machine language的概念。

今天得知那个学校的邮箱在jetbrain上没有用了，太可惜了，只能用盗版了，😭，jetbrain家的产品是真的好用，但是太可惜了。

然后马上就2022年了，还没有想好2022年怎么过，怎么去规划自己的生活，2021是一地鸡毛啊，哎。

2021年的最后一天了，领到了很多快递，以及收到了硬盘，非常开心，谢谢，希望2022加油！
